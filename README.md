# AI Agent
A simple AI Agent using Python and Google Gemini.

This implementation is based on the Boot.dev Ai Agent using Python project.

The AI Agent uses the python `uv` package management.

- install `curl -LsSf https://astral.sh/uv/install.sh | sh`
- `git clone git@github.com:ftgr/AiAgent.git`
- use `uv sync` to install the necessary dependencies described in pyproject.toml 
- `uv run main.py`
